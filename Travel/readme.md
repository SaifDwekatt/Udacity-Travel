# Travel App

## Overview
This is the **FEND Capstone Travel App BY SAIF DWEKAT**



## Installation
1. Clone this repository:
   ```sh
   git clone https://github.com/saifdwekat/Udacity-travel.git
   cd travel-app
   ```
2. Install dependencies:
   ```sh
   npm install
